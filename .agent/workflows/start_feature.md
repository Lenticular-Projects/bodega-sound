---
description: Start a new feature implementation
---

1. **Read Context**: View `project_plan.md` and `rules.md` to understand the current state and guidelines.
2. **Update Plan**: Edit `project_plan.md` to add the new feature under the "To-Do" list.
3. **Plan Implementation**: Create a new file `implementation_plan.md` outlining the steps for the feature.
4. **User Approval**: Ask the user to review the implementation plan before writing code.
